Student Grade Tracker
A simple Java console application to track and manage academic records for a class of 15 students. The program calculates overall performance and class statistics based on grades in six subjects.

Features
School Information: Capture school name, teacher's name, grade, and year of study

Student Records: Store and manage data for 15 students



Subject Tracking: Six subjects tracked per student:
-English
-Mathematics
-History
-Geography
-Science
-programming

Performance Analysis:
 -Individual student totals
 -Grade ranking (A, B, C, D, F)
 -Subject-wise averages
 -Class grade distribution

Formatted Output: Clean, tabular display of all data and statistics


How It Works
Grading System
The program calculates a total score out of 600 and assigns ranks:

A: 540+ points 

B: 480-539 points 

C: 420-479 points 

D: 360-419 points 

F: Below 360 points 



Input Process
Enter school, teacher, grade, and year information

For each of the 15 students:

Enter student name

Enter grades for all six subjects



Output Display
School header with class information

Tabular display of all student records with individual grades, totals, and ranks

Class totals and averages per subject

Grade distribution summary (count of A's, B's, etc)
